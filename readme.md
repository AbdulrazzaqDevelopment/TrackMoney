# Money Tracking Application

A C# console application for tracking personal finances by recording income and expenses organized by month.

## Features

- Track income and expenses with title, amount, and month
- Sort entries by month, amount, or title (ascending/descending)
- Filter view to show only expenses or only incomes
- Edit and remove existing entries
- Save and load data to/from file
- Text-based command-line interface

## Usage

1. Run the program
2. Add income or expense entries with title, amount, and month
3. View sorted and filtered lists
4. Edit or remove entries as needed
5. Save your data and quit (data persists between sessions)

## Requirements

- .NET Framework or .NET Core
- C# Console Application
